# Análise de Design - Dark Romance

## Paleta de Cores Extraída da Imagem

### Cores Principais:
- **Background escuro**: #0d0d0f, #1a1a1d (preto/cinza muito escuro)
- **Detalhes dourados**: #d4af37, #c9a961, #b8860b (ouro antigo)
- **Vermelho profundo**: #5c1a1a, #8b2323 (vinho/bordô das cortinas)
- **Azul noturno**: #1e2a3a, #2c3e50 (janela noturna)
- **Marrom couro**: #3d2314, #5c3317 (móveis de couro)

### Elementos Visuais:
1. **Iluminação**: Luz de velas/candelabros (efeito glow suave dourado)
2. **Texturas**: Couro capitonê, madeira escura, tecidos pesados
3. **Atmosfera**: Luxuosa, misteriosa, íntima, sofisticada
4. **Contraste**: Alto contraste entre escuro e detalhes dourados/iluminados

## Aplicação no Design:

### Tipografia:
- Fontes serifadas elegantes para títulos (ex: Playfair Display, Cinzel)
- Fonte sans-serif moderna para corpo (ex: Montserrat, Poppins)

### UI Elements:
- Cards com bordas douradas sutis
- Sombras profundas e suaves
- Botões com efeito hover de brilho dourado
- Backgrounds com gradientes escuros
- Ícones em dourado/bronze

### Efeitos:
- Box-shadows profundas
- Gradientes sutis de preto para cinza escuro
- Efeitos de brilho/glow em elementos interativos
- Transições suaves
